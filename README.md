# FITS-assignment
## Description
Flask web app to interact with the FITS file type. FITS is a file format designed to store, transmit, and manipulate scientific images and associated data.
## Rest-API endpoints
### Upload FITS file
### Write to FITS file
### Append to FITS file
### Read FITS file
### Rename FITS file
### Delete FITS file
